# 🏦 Banking System (Java)

A simple console-based Banking System developed in Java using Object-Oriented Programming (OOP) concepts. The project allows users to perform basic banking operations such as deposits, withdrawals, and balance inquiries for Savings and Current accounts through a menu-driven interface.

---

## 📌 Features

- 💰 Deposit money into Savings Account
- 💸 Withdraw money from Savings Account
- 💳 Deposit money into Current Account
- 🏧 Withdraw money from Current Account
- 📊 Check account balances
- 📋 Menu-driven console interface
- ✅ Input validation for deposits
- 🔒 Withdrawal limit for Savings Account (Maximum ₹5000 per transaction)

## ⚙️ Business Rules

### Savings Account
- Initial Balance: ₹10,000
- Maximum withdrawal per transaction: ₹5,000
- Cannot withdraw more than available balance.

### Current Account
- Initial Balance: ₹20,000
- Withdrawal allowed up to available balance.
